# Meter Manager API

## Setup

1. Install dependencies:
```bash
npm install
````

2. Create a `.env` file with your environment variables.

3. Start the server:

```bash
npx nodemon index.js
```

or

```bash
node index.js
```

## Scripts

* Lint:

```bash
npm run lint
```

* Fix lint issues:

```bash
npm run lint:fix
```

* Format code:

```bash
npm run format
```

## Database migrations (Sequelize CLI)

* Run migrations:

```bash
npx sequelize-cli db:migrate
```

* Undo last migration:

```bash
npx sequelize-cli db:migrate:undo
```
